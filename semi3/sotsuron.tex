\section*{要約}
運動する物体を我々が正確に知覚するためには神経系の遅延を補償して運動を予測する機構が必要である。この機構が実際に存在することがフラッシュラグ効果（Romi Nijhawan(1994)）の研究などによって示唆されているが、運動に速度変化が伴う場合に予測機構が受ける影響に注目した研究は少ない。Myrka Zago(2001)やRobin Baur\`{e}s (2007)らによる重力運動の予測に関する研究で重力加速度検出機構の存在について議論がなされているが、明確な結論は得られていない。一方でSchlack, A.(2008)の研究によって運動の加速度に対応して運動野の特性が一時的に変化するという報告があり、我々が何かしら速度変化を処理する機構を有している可能性をうかがうことができる。本研究ではフラッシュラグ効果を利用することで、ある加速度に特化して処理を行う機構の存在可能性を検討し、また重力方向と他方向についての差異を調査することを目的として実験を行った。その結果、加速度がフラッシュラグ効果に与える直接的な影響は現れなかったが、一部の結果について運動野の順応に関するSchlack, A.(2008)の知見を用いて一般に解釈することができ、また一部では被験者ごとに一貫した個人的な特徴が現れていることを示唆する実験結果が得られた。本研究では、ある加速度に特化して処理を行う機構の存在を否定し、速度変化を伴う運動の予測機構は重力環境などに運動野が適応することで得られる後天的な個人特有の能力であることを主張する。

\section{序論}
\subsection{フラッシュラグ効果}
我々の視覚系には網膜に像が映ってから知覚に上るまでに$\mbox{数}10 \sim \mbox{数}100\,[\mbox{ms}]$の遅延が存在する。にもかかわらず我々は運動する物体に対して的確な評価をし正確な行動を起こすことができる。このためには運動を予測する機構が必要不可欠であり、それが実際に存在することが過去の多数の研究から示唆されている。運動の先読みが行われていることを支持する現象として、一定速度で運動する刺激に対して進行方向真横に別の刺激を一瞬提示すると運動刺激に対してフラッシュ刺激が遅れて知覚されるという\underline{フラッシュラグ効果}が知られている（図\ref{FlashLag}）。この現象を発見したRomi Nijhawan(1994)は、運動検出の機能には神経系の遅延を補償するように運動を先回り予測する機構が存在する、との説明を与えた。

\subsection{重力運動の知覚}
運動を予測する機構が存在するならば、落下するボールを捕える場合のように速度が時間変化するような運動に対しても予測を行う仕組みも存在すると予想できる。自然界で観察される加速度運動は重力による落下運動であるが、Myrka Zago(2001)は重力加速度を検出する機構が存在することによって落下運動を正確に評価できるという\underline{重力内部モデル}説を提案した。しかし重力内部モデルというものを仮定しなくても、加速度運動の各瞬間速度を検出して瞬間速度と位置のみによって運動を予測することを持続的に繰り返しフィードバックすることで十分な精度で加速度運動の予測ができるので、したがって重力内部モデルが存在しなくても等速運動の予測機構と重力の知識があれば重力運動の予測は十分に可能であるという反論がある（Robin Baur\`{e}s (2007)）。
\subsection{加速度の知覚と運動野の順応}
\label{schlack}
一方で、正の加速度運動を観察しているときは減速運動を観察しているときと比較して速度変化の感度が上昇するという報告がSchlack, A.(2008)によってされている。これらのことは、MT野に存在する速度選択性細胞の応答領域が加速運動観察時により鋭くチューニングするように変化するというSchlack, A.(2007)自身の生理学的な知見で説明できると述べられている。また、加速運動を観察しているときは減速運動を観察しているときと比較して速度が過小評価されることが示されている。現在、加速度そのものに感度を持つ細胞は発見されていないが、そのような細胞が存在しないとしても、その代わりに速度選択細胞の応答特性が変化することによって加速度の評価をすることが可能であるという考察がされている。
\section{目的}
本研究では運動予測機構の影響がフラッシュラグ効果に現れることを仮定し、神経信号の遅延が現れるフラッシュラグ効果を利用することで加速度そのものを評価する機構の存在可能性を検討する。また、重力方向と他方向について加速度による影響の差異を調査することで重力加速度内部モデル仮説の妥当性を検討する。

\subsection{提示刺激}
運動円盤はフラッシュ円盤が提示される以前の区間で等加速度直線運動をし、フラッシュ提示後の区間では等速直線運動をする（図\ref{kukan}）。パラメータとして加速度$a$を変化させるが、運動円盤が画面の中心に到達した瞬間（フラッシュ円盤が提示される瞬間）にどの条件でも運動円盤の瞬間速度が$v = 16.9\,[\mbox{deg/s}]$で一定となるように初速度$v_0$および加速度$a$を調整する。ここで、初速度$v_0$を$2.178 \sim 21.78\,[\mbox{deg/s}]$まで$2.178\,[\mbox{deg/s}]$間隔で$10$段階に変化させることとする。したがって、横方向の運動については加速度が$\displaystyle a = \frac{v^2 - v_0^2}{2 \times 19.8 / 2}$（$19.8$は画面の横幅[deg]）によって$a = 14.2 \sim -9.5\,[\mbox{deg/s}^2]$の範囲で決定される。縦方向については$\displaystyle a = \frac{v^2 - v_0^2}{2 \times 16.0 / 2}$によって$a = 17.6 \sim -11.8\,[\mbox{deg/s}^2]$の範囲で決定される。運動円盤は$1\,[\mbox{pixel}]\,(=\,0.032\,[\mbox{deg}])$単位で移動する。
\paragraph{}
フラッシュ円盤が現れる場所は、フラッシュ提示時の運動円盤の位置に対して$-0.52 \sim 0.90\,[\mbox{deg}]$（運動方向が正の向き）の範囲で$0.13\,[\mbox{deg}]$間隔12通りの位置のいずれかである。フラッシュ提示時の運動円盤の速さが$v = 16.9\,[\mbox{deg/s}]$であるので、距離のずれを$v$で割ると時間タイミングずれの範囲が$8\,[\mbox{ms}]$間隔で$-31 \sim 53\,[\mbox{ms}]$となる（図\ref{kukan2}）。フラッシュ円盤の提示時間は$13\,[\mbox{ms}](=\,1\,[\mbox{frame}])$である。
\paragraph{}

\subsection{解析方法}
ある加速度および初速度の条件で、それぞれのフラッシュ提示タイミングについてフラッシュ円盤が先行して知覚された割合の5回平均を計算すると、例えば図\ref{data1}のような12点のプロットが得られる。このデータプロットをロジスティック関数
\begin{equation}
  \hat{y}_{a,\,b} = \frac{1}{1 + e^{- a \left(x - b \right)}}
\label{log}
\end{equation}
で最小二乗近似して定数$a,\,b$を決定し、その結果$\hat{y}_{a,\,b} = 0.5$となる$x$をフラッシュラグ効果による時間ずれとして扱う（図\ref{fit}）。
\begin{figure}[H]
  \centering
  \begin{minipage}{.48\textwidth}
    \includegraphics[width = \textwidth]{pic/data1.eps}
    \caption{フラッシュラグ効果の計測（例）}
    \label{data1}
  \end{minipage}
  \hspace{1zw}
  \begin{minipage}{.48\textwidth}
    \includegraphics[width = \textwidth]{pic/fit.eps}
    \caption{フィッティング（例）}
    \label{fit}
  \end{minipage}
\end{figure}
測定点を$y_i$として、残差標準偏差$\Delta y$が
\begin{equation}
  \Delta y = \sqrt{\frac{\sum_{i = 1}^{12}{\left( y_i - \hat{y}_{a,\,b} \right)^2}}{12 - 2}}
\end{equation}
で算出される。これを横軸の値へ変換するために、$\Delta y$を$\hat{y}_{a,\,b} = 0.5$となる$x$での傾き
\begin{equation}
  \left. \frac{d\,\hat{y}_{a,\,b}}{d\,x} \right|_{\hat{y}_{a,\,b} = 0.5} = \left. \frac{a e^{-a (x - b)}}{ (1 + e^{- a(x - b)})^ 2}  \right|_{\hat{y}_{a,\,b} = 0.5}
\end{equation}
で割ることで、横軸の残差標準偏差$\Delta x\,[\mbox{ms}]$が
\begin{equation}
  \Delta x = \frac{\Delta y}{\left. \frac{d\,\hat{y}_{a,\,b}}{d\,x} \right|_{\hat{y}_{a,\,b} = 0.5}} = \frac{\sqrt{\frac{\sum_{i = 1}^{12}{\left( y_i - \hat{y}_{a,\,b} \right)^2}}{12 - 2}}}{\left. \frac{a e^{-a (x - b)}}{ (1 + e^{- a(x - b)})^ 2}  \right|_{\hat{y}_{a,\,b} = 0.5}}
\end{equation}
と求まる。以後、グラフの誤差棒は残差標準偏差$\Delta x\,[\mbox{ms}]$を表すものとする。

\label{結果予想}
\label{Robin}
一般にフラッシュラグ効果は運動物体の速度が大きいほど効果が大きく現れる。仮に我々が加速度運動に特化した処理を行っておらず、Robin Baur\`{e}s (2007) が主張しているように直前$(\mbox{数}10 \sim \mbox{数}100\,[\mbox{ms}])$の瞬間速度を用いて運動の予測をしているならば、本実験において加速度を横軸に、フラッシュラグ効果を縦軸にとったデータプロットは図\ref{yosoku1}のように右肩下がりになると推測できる。一方で、加速度を含めて運動を予測する処理が行われているならば、加速度が正に大きいほどフラッシュラグ効果は大きく現れ、加速度が負に大きいほどフラッシュラグ効果は小さく現れる（図\ref{yosoku2}）というように、図\ref{yosoku1}とは逆の予想をすることもできる。
\begin{figure}[H]
  \centering
  \begin{minipage}{.48\textwidth}
    \includegraphics[width = \textwidth]{pic/yosoku1.eps}
    \caption{直前の速度のみで運動を予測した場合}
    \label{yosoku1}
  \end{minipage}
  \hspace{1zw}
  \begin{minipage}{.48\textwidth}
    \includegraphics[width = \textwidth]{pic/yosoku2.eps}
    \caption{加速度を含めて運動を予測した場合}
    \label{yosoku2}
  \end{minipage}
\end{figure}
また、Schlack, A.(2008)による心理物理学、生理学的知見(\pageref{schlack}ページを参照)によると加速度運動を観察することでMT野の応答特性が変化する。このことが本実験の結果にどのように現れるかについて一概に予想することは難しい。さらに、運動予測機構の影響がフラッシュラグ効果に現れない可能性も存在する。以上の予想が複合して結果に現れる可能性や、未知の要因が関係する可能性を考慮する必要がある。

以上のことから、フラッシュラグ効果の加速度に対する特性は左右方向間および上下方向間それぞれで類似した傾向を示すが、それは各被験者内に限ったことであって複数の被験者間の特性は異なることが示された。この結果は、被験者が各々固有の特性を有していることを示唆する。

\subsubsection{刺激運動方向間の差異}
我々が自然界で見る機会のある加速度運動は主に重力運動である。したがって鉛直下方向と上方向について加速度に対する運動予測の特性が異なることはあっても、水平方向に関しては加速度について左右の差が存在しないと考えることが合理的である。

\subsubsection{被験者間の差異}
フラッシュラグ効果の左右方向間および上下方向間の差を加速度間で平均した値や、重力方向と水平方向との差異について、すべての被験者に共通する性質は見られない（表\ref{sabun2}）。このことは\pageref{結論２}ページの結果のように被験者が各々固有の特性を有することと同様である。
\subsection{先行研究との関連}
\begin{wrapfigure}{r}{.53\textwidth}
  \vspace*{-\intextsep}
  \includegraphics[width = .53\textwidth, clip]{pic/sch.eps}
  \caption{速度選択性細胞の順応による運動野の特性変化}
  \label{sch}
%  \vspace*{-\intextsep}
\end{wrapfigure}
Schlack, A.(2008)による心理物理学、生理学的知見(\pageref{schlack}ページを参照)は図\ref{sch}のようなモデルで表現できると著者は考える。図\ref{sch}の曲線は物理的な速さと知覚上の速さの対応を示しており、減速運動を観察している場合と加速運動を観察している場合とで対応関係の特性が変化することを示している。そして、曲線の傾きが大きい部分では速度を知覚するための資源が物理的な速度の狭い範囲に集中して割り当てられているために速度変化の感度が高いことを意味する。逆に傾きが小さい部分では少ない資源で大きな範囲を扱うために速度変化の感度が低いことを意味する。図\ref{sch}によると、正の加速運動を観察しているときは現在観察している物理的速度に対応する領域で速度変化感度が上昇し、またその領域に対応する知覚的速度が低域にシフトすることで速度が過小評価される。逆に、減速運動を観察しているときは速度変化感度が低下する代わりに速度が過大評価されるように特性曲線がシフトする。
\paragraph{}
運動物体の物理的速度が大きいほどフラッシュラグ効果が大きく現れる（\pageref{Robin}ページ参照）ということが知覚的速度についても同様ならば、本実験において正の加速度運動を提示した場合は減速運動を提示した場合と比較して図\ref{sch}のように速度が過小評価されることでフラッシュラグ効果が小さく現れるであろう。図\ref{ave}を見ると、提示加速度が約$-5\sim5\,[\mbox{deg/s}^2]$の範囲でフラッシュラグ効果$[\mbox{ms}]$が予想と一致して右肩下がりである。しかし提示加速度が大きくなるとその傾向は消失する。\pageref{有意でない}ページで述べたように被験者間で有意に結果が類似していないので図\ref{ave}のグラフの形状に基づく議論をすることは一般にできないが、しかし被験者間で結果が類似しないことを一部否定する図\ref{R_ave}や図\ref{U_ave}のような証拠も存在する。
\subsubsection{結果の解釈}
図\ref{R_ave}$\sim$図\ref{U_ave}のように被験者間で結果が類似することを支持する場合と支持しない場合の両方が存在することの原因として、提示加速度に対するフラッシュラグ効果の変化には一般に共通した傾向と個人差との両方が含まれるという可能性が考えられる。そして、その個人差のフラッシュラグ効果に現れる度合いが提示加速度の領域によって変化すると仮定すると、図\ref{ave}のグラフにばらつく領域とばらつかない領域が存在する原因は約$-5\sim5\,[\mbox{deg/s}^2]$で一般に共通した傾向が相対的に大きく、他の領域で個人差が相対的に大きく現れていることであるという推測ができる。また、加速度が正に大きい領域で個人差が相対的に大きく現れることは、図\ref{sch}のモデルのように速度変化感度が上昇することと関連すると考えることができる。
\paragraph{}
運動野の順応に個人差があることはよく知られている。本研究の結果は、加速度運動の処理も同様に速度選択性細胞が順応することによって行われている可能性を示唆する。重力方向との差異に関して、方向間で差異は本来存在せずに重力環境に適応することで順応に個人差が生じると考えれば、例えば落下するボールの軌道を正確に予想することが練習によってできるようになることの説明が与えられる。したがって、必ずしも$9.8\,[\mbox{m/s}^2]$に特化した予測機構が生まれつき存在すると考えることはできない。本研究の実験環境では視距離が$50\,[\mbox{cm}]$、提示された運動の加速度が最大で$17.6\,[\mbox{deg/s}^2]$であり、これを$[\mbox{m/s}^2]$に換算すると$0.16\,[\mbox{m/s}^2]$である。現実の重力加速度を視距離$50\,[\mbox{cm}]$で再現しようとすると運動が速すぎて実験を行うことができないためにこのような設定をした。実環境で重力加速度が$9.8\,[\mbox{m/s}^2]$一定でも視距離によって$[\mbox{deg/s}^2]$は変化する。さらに、実世界の加速運動はほぼすべて厳密に等加速度運動ではない。よって、一定の重力加速度に特化して応答を示す細胞というものを仮定しても無意味である。
\section{結び}
\subsection{結論}
本研究では加速度を伴う運動予測機構の影響がフラッシュラグ効果に現れることを仮定して実験を計画した。その結果、\pageref{結果予想}ページの予想（図\ref{yosoku1}、図\ref{yosoku2}）のように加速度の影響が直接的に現れることはなかった。しかし一部の加速度の範囲について運動野の順応に関する先行研究の知見を用いて一般に解釈することができ、また他の領域では被験者ごとに一貫した個人的な特徴が現れていることを示唆する結果が得られた。
\paragraph{}
したがって、速度変化を伴う運動の予測機構は経験や環境に運動野が適応することで得られる後天的な個人特有の能力であり、重力方向と水平方向との差異があるならばそれは重力運動に適応した結果であると考えられる。
\subsection{今後の課題}
本研究では運動予測機構に加速度が与える影響を考察するためにフラッシュラグ効果という間接的な手法を用いたが、今後は representational momentum などのパラダイムを用いて直接的な効果を調べる必要がある。また、奥行方向に運動する物体の位置予測について等方性が成立しないことが知られており、このことについて加速度運動と関連して研究する余地がある。

\subsection*{系列の類似性検定}
本研究において著者が独自に考案した、二つの系列の類似度を判定するアルゴリズムを以下に示す。
\begin{itemize}
  \item 二つの系列を$a_i,\,b_i\,(i = 1,\,2,\,\cdots,\,n)$とする
  \item 検定量を$\displaystyle r = \sum_{i = 1}^n{\left| a_i - b_i \right|}$とする
  \item $i = 1,\,2,\,\cdots,\,n$のうち任意の$i$を$rand$として$\displaystyle x = \sum_{i = 1}^n{\left| a_{rand} - b_{rand}\right|}$を計算する
  \item 十分に小さい$p\,\,(p \ll 1)$に対して、$x$を十分な回数繰り返し計算して得られる$x$の度数分布のうち$100(1 - p)\,[\%]$が検定量$r$から右側に存在するとき、\underline{系列$a_i,\,b_i$は有意に類似する}と定義する。
\end{itemize}
